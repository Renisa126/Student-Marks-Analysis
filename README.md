# Student Performance Analysis

## 📌 Overview
This project analyzes student performance using Python by calculating average marks and identifying top-performing students.

## 🎯 Objective
To understand student performance patterns using data analysis techniques.

## 📊 Dataset
- Student marks in Maths, Science, English

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## 🚀 Features
- Calculated average marks
- Sorted students based on performance
- Identified top-performing students
- Visualized data using graphs

This analysis helps in understanding student performance trends and identifying strong and weak areas.
